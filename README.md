# tuneblend

Intelligently sort your setlist using nothing more than a streaming service's playlist.

## Getting Started

### Dependencies

* [yarn package manager](https://yarnpkg.com/getting-started)
* Node.js environment

Install via homebrew
```bash
 brew install yarn
```

## Development

```bash
yarn create electron-app tuneblend
```
